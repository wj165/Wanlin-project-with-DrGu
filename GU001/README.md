# GU001-kidney
GU001: Kidney histology dataset split and baseline experiments


## Dataset splits

The file `data/kidney_Papillary_vs_Clearcell_split.xlsx` is a **derived train/validation/test split**
generated from `kidney MetaData_Release_1.csv` using the notebook in `notebooks/`.

This file represents the exact split used for GU001 (Papillary RCC vs Clear Cell RCC),
and is not an independent or supplementary raw dataset.
